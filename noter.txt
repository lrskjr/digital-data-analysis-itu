# Basic intoroduction til text mining
Hvorfor er interessant for digital design? 
- Text mining preprocessing: Cleaning : full stop, symbols and stopwords (noise).
     Normalization: Lowercasing, Punctuation removal, optional is stemming ( reducing to a kind of root - not grammatical root ) and
     lemmatization (a grammatical root) . 
     
     Tokenization: breaking into sentences, words, and letters.
     
     Filtering af stopord: exclusion of recurring meaningless words.
     
     
     
- Text data exploration: Word clouds og andre visualiseringer, frequency tables.
- Text data representation: vectorizing , embedding for stastistical stuff.
- Feature engineering - extract numbers from data: N-grams, pos-tagging
- Sentiment Analysis: Spotting expressed feelings in text (positive, neutral, anger-related, anxiety-related ect.)
- Text clustering and topic modeling: to spot the unddddderlying patterns inside a set of documents - labelled unsuperviced learning.
- Word Frequency table - word cloud 
- Network plot a word co-occurence network
